# Layout Simulator - CLAUDE.md

## プロジェクト概要

2Dの平面上にアイテムを自由に配置できるレイアウトシミュレーターWebアプリ。
キャンプ場のテント配置・部屋のインテリア配置などのユースケースを想定。
Webブラウザ動作、スマホ・PC両対応のレスポンシブデザイン。

## 開発ガイドライン

### 計画フェースのルール

<important>
- 実装前に必ずプランモードで設計を確認する
- 大きな変更はフェーズ分けしたゲート式プランを作成する
- 不明点は AskUserQuestion ツールで確認してから進める
</important>

### コーディングルール

- シンプルさを優先：1つの関数に1つの責務
- 投機的な抽象化をしない（実際に必要になるまで汎用化しない）
- エラーハンドリングはシステム境界（ユーザー入力・外部API）のみ
- コメントはロジックが自明でない箇所にのみ追加

### Git コミットルール

<important>
- ファイル単位で個別コミットを作成する（複数ファイルを1コミットにまとめない）
- コミットメッセージは変更の「なぜ」を説明する
- 作業ブランチ: `claude/layout-simulator-requirements-cJdxY`
- main ブランチには直接プッシュしない
</important>

### コンテキスト管理

- コンテキスト使用率50%で `/compact` を実行する
- CLAUDE.md は200行以内に収める
- 大きなルールセットは `.claude/rules/` に分割する
