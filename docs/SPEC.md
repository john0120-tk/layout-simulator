## 1. 画面一覧

```
┌─────────────────────────┐
│      ホーム画面          │  ← レイアウト一覧・管理
└────────────┬────────────┘
             │ レイアウトを選択 or 新規作成
             ▼

└─────────────────────────┘
```

## 2. ホーム画面

### 目的
作成済みのレイアウトを一覧表示・管理する画面。

### レイアウト

```
┌────────────────────────────────────┐
│  Layout Simulator          [+ 新規] │  ← ヘッダー
├────────────────────────────────────┤
│                                    │
│  ┌──────┐ ┌──────┐ ┌──────┐      │
│  │ サム │ │ サム │ │ サム │      │
│  │ネイル│ │ネイル│ │ネイル│      │
│  │      │ │      │ │      │      │
│  │春キャ│ │夏キャ│ │新規  │      │
│  │ンプ  │ │ンプ  │ │      │      │
│  └──────┘ └──────┘ └──────┘      │
│                                    │
└────────────────────────────────────┘
```

### 機能
- レイアウトカードをクリック → エディタ画面へ
- `[+ 新規]` → 名前入力ダイアログ → エディタ画面へ
- カード上で右クリック or 長押し → 「名前変更」「削除」メニュー
- データ保存先：ブラウザのローカルストレージ

## 4. 各パネル仕様

### 4-1. ヘッダー

| 要素 | 内容 |
|------|------|
| `← 一覧` | ホーム画面に戻る |
| レイアウト名 | クリックで名前編集 |
| `エクスポート` | PNG画像でダウンロード |
| `設定` | フィールドサイズ・表示オプション設定 |

---

### 4-2. レイヤーパネル（左）

レイヤー構造とアイテム一覧を表示・管理。

```
┌──────────────────┐
│ レイヤー         │
├──────────────────┤
│ ▼ フィールド     │  ← 展開/折りたたみ
│   ▼ テント A     │
│     ■ シュラフ   │  ← アイテム
│     ■ ランタン   │
│   ▼ テント B     │
│     ■ マット     │
│   ■ テーブル     │  ← フィールド直置き
│   ■ チェア       │
├──────────────────┤
│ [+ アイテム追加] │
└──────────────────┘
```

**操作**
- アイテム名クリック → 選択状態（キャンバス上でもハイライト）
- ドラッグ → レイヤー間の移動（ギアをテント内/外に移動）
- アイテム右クリック → 「複製」「削除」

---

### 4-3. キャンバス（中央）

実寸比率で描画されるメインのシミュレーションエリア。

**表示要素**
- フィールド（最背面、グレーの枠）
- テント（中間層、色付きの図形）
- ギア（最前面、色付きの図形）
- 寸法ラベル（表示/非表示 切替可）
- グリッド目盛り（表示/非表示 切替可）

**スケール変換**
- フィールドサイズと画面サイズから自動計算
- 例：フィールド 1000cm → 画面 800px = 1px あたり 1.25cm

**操作**
- ホイールスクロール / ピンチ：ズームイン・アウト
- 中ボタンドラッグ / 2本指ドラッグ：キャンバス移動（パン）
- アイテムクリック：選択（右パネルに詳細表示）
- アイテムドラッグ：移動
- 選択中アイテムの回転ハンドル（上部）：ドラッグで回転
- 空白クリック：選択解除

---

### 4-4. アイテムプロパティパネル（右）

選択中のアイテムの詳細設定。アイテム未選択時はフィールド設定を表示。

#### アイテム選択時

```
┌──────────────────┐
│ テント A         │  ← 名前（編集可）
├──────────────────┤
│ 形状  [長方形 ▼] │
│ 幅    [230] cm   │
│ 高さ  [300] cm   │
│ 角度  [ 45] °    │
│ 色    [■ ▼]      │
│ レイヤー[テント▼]│
├──────────────────┤
│ [複製] [削除]    │
└──────────────────┘
```

#### アイテム未選択時（フィールド設定）

```
┌──────────────────┐
│ フィールド設定   │
├──────────────────┤
│ 幅   [1000] cm   │
│ 高さ [ 800] cm   │
├──────────────────┤
│ 表示設定         │
│ ☑ 寸法ラベル     │
│ ☑ グリッド目盛り │
│ □ グリッドスナップ│
└──────────────────┘
```

### 4-5. ツールバー（下）

| ボタン | 機能 |
|--------|------|
| `[+ アイテム]` | アイテム追加ダイアログを開く |
| `[選択]` | 選択・移動モード（デフォルト） |
| `[計測]` | アイテム間の距離計測モード |

## 6. データ構造

```typescript
// レイアウト
type Layout = {
  id: string;
  name: string;
  field: Field;
  items: Item[];
  createdAt: string;
  updatedAt: string;
};

// フィールド（キャンプサイト）
type Field = {
  widthCm: number;
  heightCm: number;
};

// アイテム（テント・ギア共通）
type Item = {
  id: string;
  name: string;
  type: 'tent' | 'gear';        // レイヤー種別
  parentId: string | null;      // テント内ギアはテントのID、それ以外はnull
  shape: 'rect' | 'triangle' | 'diamond' | 'pentagon' | 'hexagon' | 'circle';
  widthCm: number;
  heightCm: number;
  x: number;                    // フィールド上の位置（cm）
  y: number;
  rotation: number;             // 度数
  color: string;                // hex
};

// アプリ全体の状態
type AppState = {
  layouts: Layout[];
  displayOptions: {
    showDimensionLabels: boolean;
    showGrid: boolean;
    snapToGrid: boolean;
  };
};
```

## 7. 画像エクスポート仕様

- 形式：PNG
- 内容：キャンバスの表示内容（UI要素・パネルは含まない）
- ファイル名：`{レイアウト名}.png`
- 解像度：現在の表示サイズ（将来的に高解像度オプションも検討）
